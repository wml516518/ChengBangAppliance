# 诚邦家电维修管理系统

基于 .NET 8 + ASP.NET Core MVC + SqlSugar + SQLite 的简易家电维修管理程序。

## 功能

- **用户/管理员注册**：支持普通用户与管理员注册（注册时勾选“注册为管理员”即为管理员）
- **登录/退出**：Cookie 认证，支持“记住我”
- **商品列表**：首页可浏览商品，支持“仅看上架”
- **商品管理**（仅管理员）：添加、编辑、删除商品；**上架/下架**一键切换

## 运行

1. 安装 [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
2. 在项目目录执行：
   ```bash
   dotnet restore
   dotnet run
   ```
3. 浏览器打开 http://localhost:5000

首次运行会自动创建 SQLite 数据库 `appliance.db`（位于项目根目录）。

## 使用说明

1. 先访问 **注册**，填写用户名（至少2位）、密码（至少6位），若需管理员权限可勾选“注册为管理员”
2. **登录** 后，管理员可在导航栏进入“商品管理”，进行商品的增删改与上下架
3. 所有用户可在“商品列表”查看商品，并可筛选“仅看上架”
